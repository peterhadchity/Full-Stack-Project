About this project

Why Roast

Roast is a small field guide to coffee — the origins worth knowing and what actually makes them taste the way they do. It pairs a curated collection with a live nutrition lookup for everyday drinks.

How it's built

Hand-written semantic HTML5 and CSS3 with Flexbox, a little Bootstrap 5 for the grid, and vanilla JavaScript organised into ES6 classes (CoffeeApp, Modal, Collection, NutritionAPI, Lookup) — no jQuery. The Collection opens a hand-built detail modal; the Nutrition page fetches from the API Ninjas Nutrition endpoint with client-side search over the results.

About the data

The collection's tasting notes are written for this project. Live nutrition figures come from API Ninjas.

Who made it

Built by Peter Hadchity as the final project for Full Stack Development, Lebanese University — Faculty of Engineering, 2026.
